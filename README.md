# Docker Compose 
## for PostgreSQL 15.4 and PgAdmin 7.6

### Run command for start server
In folder the docker-compose.yml
```
docker compose up -d
```

### Run command for stop server
In folder the docker-compose.yml
```
docker compose down
```
